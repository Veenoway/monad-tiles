## Getting Started

### 1. Clone the Repository
```bash

```

### 2. Install Dependencies
```bash
npm install
# or
bun install
```

### 3. Run Development Server
```bash
npm run dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.
